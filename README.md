# 💎 CleanCut Image (인물 알파 추출 및 이미지 편집)

> **"삼국지 게임에서 사용되던 알파채널"**
>
> 즐겨하던 삼국지게임에서 신무장을 등록할 시에 배경을 제거하여 인물만 보여지고 삼국지 배경이 보이도록 해야했습니다. 스스로 포토샾을 통해 알파를 따는 방법을 익히기도 하였지만 시간이 오래걸리기도 하고 복잡한 이미지는 따기 어렵기까지 했습니다. 또한 삼국지 카페에서도 요청이 많았고 요새는 ai로 배경을 제거하는 방법을 아는 사람들이 많아져서 요청이 줄어들긴 했으나, 여전히 요청하는 사람들과 이미지 조절도 부탁하는 요청이 보여 웹서비스를 만들게 되었습니다.
제가 머리 속으로 설계한 바와 같이 구현되었고 배포까지 고려하였습니다.

---

## 🎨 주요 기능

* 📏 리사이징: 삼국지 시리즈별 규격에 맞춘 크기 조절 및 미리보기.

* ✂️ 크롭: 드래그 앤 드롭 방식의 자유로운 영역 추출 및 미리보기.

* 👤 AI 인물 추출: SOTA 모델 기반의 정밀 배경 제거 및 알파 채널 생성.

* 🖌️ 마스크 보정: AI가 놓친 세밀한 경계면을 사용자가 직접 브러시로 복구/삭제.

* 💾 데이터 다운로드: 모든 편집 단계에서 원본 포맷은 물론, 알파 채널이 포함된 PNG 및 BMP 포맷으로 즉시 다운로드 가능.

## 🏗️ 핵심 설계 내용

설계와 프로세스는 오로지 저의 의지에 따라 100% 구현되었으며 저의 손 코딩이 들어가지 않고 AI활용을 위해 100% AI에이전트에게 명령을 내려 약 3일내에 만든 프로젝트 입니다.

### 1. 이미지 프로세스
* 리사이즈 - 크롭 - 알파 추출의 순서로 원하는 이미지를 만들어가도록 하였으며 각 단계에서도 미리 알파 추출을 하였더라도 알파가 적용된 이미지를 다운받거나 원래 포맷에서 다운로드 할수 있도록 하였습니다.

### 2. 이미지 미리보기
* 이미지를 여러번 편집할 경우 이미지의 화질이 저하될 수 있으므로 원본 이미지를 유지하면서 미리보기를 할 수 있도록 하였습니다.

### 3. 알파 추출 후
* 알파 추출 후 배경과 인물이 어중간하게 있어 추출이 잘 안될 경우를 대비하여 하단에 마스크 적용기능을 추가하여 배경이 포함되어있는 부분을 지우거나 인물에 포함됨에도 불구하고 포함되지 못하는 부분을 포함할 수 있도록 하는 기능을 추가하여 원하는 이미지를 얻을 수 있도록 하였습니다.

### 4. 배포
* 배포 시에는 한 사람이 무분별하게 사용하지 못하도록 ip당 5장, 총 100장으로 제한을 두었습니다.
* gpu모델을 이용함으로 aws 사용 시 응답이 필요한 경우에만 사용될 수 있도록하고, 사용하지 않을 때는 서버를 종료하여 비용을 절감하도록 할 예정이였으나 배포는 하지 않았습니다.

## 🤖 AI 기술 분석 및 평가 (by Antigravity Gemini 3.1 Pro)

본 프로젝트의 코드 베이스를 AI 에이전트가 정밀 분석한 결과입니다. (수정 전 상태의 객관적 리포트)

### ✅ 기술적 강점 (Pros)
* **그래픽스 수학 알고리즘**: 브러시 시스템의 `falloff = 1 - t² * (3 - 2t)` 공식(Smoothstep) 적용으로 전문 그래픽 툴 수준의 자연스러운 경계면 처리 구현.
* **고성능 SPA 구조**: Vanilla TS와 Custom Router를 통해 프레임워크 오버헤드 없는 즉각적인 상태 관리 및 화면 전환 실현.
* **픽셀 단위 제어**: `ImageData` 배열 직접 조순회를 통한 픽셀 데이터 연산으로 대용량 비트맵 처리 시 최상의 반응 속도 확보.
* **실무형 백엔드**: IP 기반 Rate-Limit 및 DBS(Day-Based-Storage) 로깅 시스템 등 상용 SaaS 급의 리소스 방어 체계 구축.

### ⚠️ 기술적 개선 제안 (To-be Optimized)
* **렌더링 최적화**: 현재 `innerHTML` 기반의 구역 재렌더링 방식을 DOM 직접 조작 방식으로 변경하여 브라우저 리플로우(Reflow) 최소화 필요.
* **에러 핸들링**: 네트워크 지연이나 유효하지 않은 요청에 대한 사용자 피드백(UI Toast 등) 강화 제안.
* **보안 강화**: 서버측 입력값 검증 로직의 세분화 및 예외 처리 로직 보강 필요.
